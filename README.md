# Personal Decision Memory Engine

## Overview
The Personal Decision Memory Engine helps users record their decisions, reasons, and outcomes.  
By analyzing past decisions, users can improve future decision-making.

## Features
- Add personal decisions
- Store decision history
- Update actual outcomes
- View all decisions

## Technologies Used
- Python
- JSON

## How to Run

1. Clone the repository

git clone https://github.com/yourusername/personal-decision-memory-engine

2. Navigate to the folder

cd personal-decision-memory-engine

3. Run the program

python decision_engine.py
